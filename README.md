# 🎬 Netflix Landing Page Clone

A responsive Netflix landing page clone built using **HTML5** and **CSS3**. This project recreates the look and feel of the Netflix homepage while focusing on responsive layouts and modern web design.

## 🚀 Features

- Responsive landing page
- Netflix-inspired user interface
- Modern CSS styling
- Clean and organized code structure

## 🛠️ Tech Stack

- HTML5
- CSS3

## 📂 Project Structure

```
Netflix_clone/
│── index.html
│── style.css
│── logo.png
│── back.png
│── poster1.jpg
│── poster2.jpg
│── poster3.webp
│── poster4.avif
│── poster5.png
```
